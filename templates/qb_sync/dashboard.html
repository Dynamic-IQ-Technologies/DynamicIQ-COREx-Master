QuickBooks Online Setup
  1. Go to developer.intuit.com and create a free developer account.
  2. Create a new app — select QuickBooks Online and Payments.
  3. Under Keys & credentials, copy your Client ID and Client Secret.
  4. Add the following Redirect URI in your app settings:
    {{ request.host_url }}qb/callback
  5. Return to this page and enter the Client ID and Client Secret in the QuickBooks App Credentials panel at the top. Click Save Credentials.
  6. Click Connect to QuickBooks in the Connection panel to complete the sign-in.
QuickBooks Desktop (Polling Mode)
  1. Install the QuickBooks Web Connector on the machine running QuickBooks Desktop.
  2. In your QB Desktop settings, enable Web Services and note the service URL.
  3. Enter the Web Connector polling URL in the Desktop Poll URL field (not shown in online mode).
  4. The sync engine will poll QB Desktop on a scheduled interval to push/pull data.
  5. Desktop sync events are logged identically to Online events in the Sync Log tab.
QB Desktop sync requires network access from this server to the QB Desktop machine.
Security Notes
  • OAuth tokens are stored in the database and refreshed automatically before expiry.
  • In production, ensure QB_SANDBOX=false is set and your QB app is approved for production.
  • Webhook signatures are verified using HMAC-SHA256 against the intuit-signature header.
  • Only Admin and Accountant roles can connect, configure, or disconnect the integration.
