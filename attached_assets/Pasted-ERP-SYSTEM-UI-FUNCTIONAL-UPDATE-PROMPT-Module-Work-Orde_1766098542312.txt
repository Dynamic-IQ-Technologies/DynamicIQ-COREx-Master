2. Task-Level Material Requirements

Enhance the Work Order Tasks section to allow materials to be assigned per task, not just at the work order level.

Material Requirement Fields

Each task must allow adding multiple material lines with:

Field	Requirement
Part Number	Required
Description	Auto-populated
Required Quantity	Required
Issued Quantity	System-tracked
Unit of Measure	Required
Lot / Serial Number	Optional
Warehouse / Location	Required
Material Status	Planned / Issued / Consumed
Required By Date	Optional
3. Material Behavior & Logic

Materials added at task level must:

Be consumed when task is completed

Prevent task completion if required materials are not issued (configurable)
