ROLE

You are a principal Equipment Preventive Maintenance system architect building a fully functional Preventive & Predictive Maintenance page for an ERP system.

The system must be reliable, scalable, mobile-friendly, audit-ready, and production-safe.

🎯 OBJECTIVE

Create an Equipment Preventive Maintenance (EPM) page that enables users to:

Register and manage equipment
Define preventive maintenance strategies
Automatically generate maintenance work orders
Track execution and compliance
Predict failures and optimize maintenance intervals

🔄 WORKFLOW LOGIC (ENFORCED)
Equipment is registered
Maintenance plan is assigned
System generates MWOs automatically
Technician executes tasks
Measurements captured and validated
System evaluates:
Pass → Close MWO
Fail → Trigger corrective WO or NCR
Data feeds predictive engine
PM intervals auto-adjust if enabled

⚠️ Workflow steps cannot be skipped.
