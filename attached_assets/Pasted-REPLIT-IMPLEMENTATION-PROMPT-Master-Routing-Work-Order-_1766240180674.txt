🧩 CORE MODULE CONCEPT

Master Routing = Reusable Work Order Blueprint

Each Master Routing represents:

A standard process to build, inspect, repair, or overhaul a part

A template that can auto-generate Work Orders and tasks

Routing Name

Manufacturing

Inspection

Overhaul

Calibration

Part Number (or Part Class / Category)

Revision

Status (Draft, Approved, Obsolete)

Default Work Order Type

Effective Date / Expiration Date

Regulatory Basis (FAA, EASA, AS9100, Internal)

Created By / Approved By

Timestamps

Create MasterRoutingOperation linked to MasterRouting:

Operation Sequence Number

Operation Name

Description / Instructions

Work Center / Department

Standard Labor Hours

Skill / Certification Required

Tooling / Equipment Required

Setup Time & Run Time

Operation Type:

Build

Disassembly

Inspection
